Todo en Flutter, son Widgets.

Widget: Es una clase que puede tener argumentos posiciones y argumentos con nombre.

Ejemplo:
AppBar( title: Text('Hola Mundo'), )

class Text{ String mensaje; Text(this.mensaje); }

class AppBar{ Text title; AppBar({ this.title }); }

Estados de los Widgets
StatelessWidget StateFullWidget
Sin estado Con estado

Arbol de Widgets: widgets dentros de otros, todos parten de un principal.
